# mini-library-project
this is a mini library project that me and my friend with some support made in our high school.


please before running the web app do these requisites:

create mysql database named "library".
create table inside that db named "student","admin","booklist".
create columns for each of the tables as follows:

        student=
        slno  INT,
        name TEXT 50,
        clgname TEXT 50,
        branchname TEXT 50,
        contact_no INT 10,
        email TEXT 20,
        password VARCHAR 20

        booklist=
        id INT,
        book_name TEXT 50,
        author_name TEXT 50,
        date_of_publication DATE


        admin=
        slno INT,
        username TEXT 50,
        password VARCHAR 20
